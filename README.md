*What is it?*
-----------
Oxford Independent Health is a new way of looking at Private Healthcare.  

*Installation*

This software is designed to run with Ruby 2.2+ and Rails 4+

To install dependencies
````
bundle!
````
To run the tests
````
rake spec
````
To start the server
````
rails s
````
